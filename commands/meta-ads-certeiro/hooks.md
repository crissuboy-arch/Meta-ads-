Gere hooks seguindo a skill `hooks-criativos` na íntegra.

## Passos

1. Determine o produto e quais formatos o usuário quer (Feed, Stories, Reels, UGC, Carrossel, vídeo curto — ou todos).
2. Puxe dores/desejos/objeções reais de `mercado-[produto].md` e/ou `pesquisa-[produto].md`, se existirem. Se nenhum dos dois existir, pergunte ao usuário as 2-3 dores principais do público antes de gerar — hook sem dor real vira genérico.
3. Para cada formato pedido, gere 3-5 hooks usando padrões diferentes da skill (pergunta, choque, identificação, erro comum, resultado, confissão).

## Exemplo de uso

```
/hooks clareamento dental a laser — Reels e Stories
```

## Saída

Liste os hooks organizados por formato, marcando de qual dor/desejo cada um partiu. Sugira o próximo passo: `/ugc [produto]` se o formato incluir UGC/vídeo, ou `/teste-ab [produto]` para priorizar quais hooks testar primeiro.
